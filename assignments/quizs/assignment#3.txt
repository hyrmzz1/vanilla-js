function(;함수) 이란 코드를 encapsulate(;캡슐화) 하는 것. => 반복 가능!
함수를 실행하는 방법은 nameOfFunc() 	// () 붙여주기

함수에서 arguments(;인수)를 사용하는 이유는 함수로 값을 보내기 위함.
인수 개수는 제한 없음.(없어도 된다) 단, 인수 순서대로 데이터 잘 넣어주기.

함수에서 return 을 사용하는 이유는 함수로부터 값을 가져오기 위함.
=> 함수 외부에서도 return 입력한다면 호출 시 그 값 출력되어 사용 ㄱㄴ하게 된다.
+) return 이후에 작성된 코드는 작동하지 않음.

parseInt() 함수는 string을 int로 바꿔줌. parseInt() 에 숫자로 변환할 수 없는 문자 들어갈 경우(ex. "hello") NaN 이 return 된다.
